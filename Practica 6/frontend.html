El frontend.

Esta es la parte de una aplicación o sitio web con la que los usuarios interactúan directamente. Incluye elementos como la disposición de la página, los botones, los formularios, los menús desplegables y cualquier otro componente que los usuarios puedan ver y con los que puedan interactuar. Los desarrolladores front-end utilizan lenguajes de marcado como HTML para estructurar el contenido, hojas de estilo como CSS para darle estilo y JavaScript para hacer que la página sea interactiva. Se refiere a la parte visible de un sitio web o aplicación, es decir, la interfaz que los usuarios ven y con la que interactúan. La función del frontend es presentar la información de manera clara y atractiva, y permitir a los usuarios interactuar con la aplicación de manera intuitiva.
